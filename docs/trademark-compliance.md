# Trademark Compliance

Arkived is an independent open-source project. It is not affiliated with, endorsed by, or sponsored by Microsoft Corporation.

"Microsoft," "Azure," "Microsoft Azure Storage Explorer," and related marks are trademarks of Microsoft Corporation.

This document explains how Arkived complies with [Microsoft's General Trademark Guidelines](https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks).

## What we do

- **We clearly state Arkived is independent.** Every surface (README, website, crate descriptions, app metadata) makes the independence explicit.
- **We truthfully describe compatibility.** We say "works with Microsoft Azure Storage" — this is allowed under Microsoft's guidelines.
- **We use Microsoft marks as adjectives, not nouns.** "Microsoft Azure Storage" (adjective form), not "Azure" standalone as if it were ours.
- **We use the full product name on first reference.** "Microsoft Azure" rather than just "Azure" at the top of docs.
- **Our own brand is more prominent than Microsoft's** in all marketing materials.
- **We do not use Microsoft logos** anywhere — not in the app icon, README, website, social media, or binaries.
- **We do not imply partnership or endorsement.**

## What we do not do

- **We do not name our product after any Microsoft product.** "Arkived" is not derived from any Microsoft mark.
- **We do not register any trademark, domain, or handle** that is identical or confusingly similar to Microsoft's marks.
- **We do not reproduce Microsoft's Storage Explorer UI, icons, or visual identity.**
- **We do not use any logo, icon, or design element from Microsoft** unless and until we have an express license.
- **We do not distribute Microsoft binaries, assets, or documentation.**

## Guidelines for contributors

When contributing documentation, marketing content, blog posts, or social media content:

1. Do not begin any heading, title, or product name with a Microsoft mark.
2. Prefer "Arkived is a storage client for Microsoft Azure" over "Arkived is an Azure storage client."
3. When comparing Arkived to Microsoft's products, be factual and neutral.
4. Do not use Microsoft logos or brand colors.
5. If in doubt, ask in a PR review before merging.

## Attribution in the README

The README includes the following canonical attribution:

> Arkived is an independent open-source project. It is not affiliated with, endorsed by, or sponsored by Microsoft Corporation. "Microsoft," "Azure," and "Microsoft Azure Storage Explorer" are trademarks of Microsoft Corporation.
